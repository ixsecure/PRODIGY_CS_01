# Task 01 - Caesar Cipher Implementation

## Description
Python program that encrypts and decrypts text
using the Caesar Cipher algorithm.

## Features
- Encrypt a message with a custom shift value
- Decrypt an encrypted message
- Handles uppercase and lowercase letters
- Preserves spaces and special characters

## How to Use
1. Run : python cesar.py
2. Choose 1 (encrypt) or 2 (decrypt)
3. Enter your message
4. Enter the shift value

## Example
- Message : hello | Shift : 3 | Result : khoor
